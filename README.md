
  # Farmer-Labour Machine Finder

  This is a code bundle for Farmer-Labour Machine Finder. The original project is available at https://www.figma.com/design/Jnod58YsrBDzKzz11kFsCW/Farmer-Labour-Machine-Finder.

## Running the code

Run `npm i` to install the dependencies.

Run `npm run dev` to start the development server.

## Demo Scope (Front-end only)

- Fully functional UI prototype with mock data and localStorage.
- Role-based dashboards: Farmer, Labourer, Machine Owner, Admin.
- Smart matching, map, AI recommendations, weather alerts are UI-only placeholders.
- Payments, OTP, Aadhaar e-KYC, GPS, and integrations are planned for future scope.
